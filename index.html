Правила пользования сайтом
Данный сайт занимается образовательной некоммерческой деятельностью, цель которой дать полезную информацию всем желающим. Большая часть материалов содержат ссылки на ресурсы Интернет, которые помогут познакомиться с дополнительными материалами по интересующей Вас теме. Распространение ресурса и копирование информации приветствуется! :D
Обзор сайта
1. Навигация.

Навигационная панель - одна из самых важных элементов на сайте. В ней содержится только самая полезная информация и информация, предназначенная для быстрого доступа

2. Ссылки.

Основной архив ссылок. Именно здесь лежит вся тематическая информация о технологиях, например, все, что касается Веб-разработки. В каждой из тем - куча заметок.

3. Цвета HTML.

Удобная таблица для верстальщиков, дизайнеров и всех причастных к веб-разработке.
Формат таблицы: ИМЯ ЦВЕТА В HTML : HEX : RGB

4. Специальные символы
Специфическая таблица с дополнительной навигацией для верстальщиков на случай, если нужно добавить на сайт что-то тематическое, например: Знаки математических операций.
5. Карты
Место работы Владельца сайта. Очень престижное заведение. А так, бесплатное импортированное API Яндекса через Maps Constructor, полезная фича, пользуйтесь!
